1. Set an environment variable SAHI_HOME pointing to your SAHI home directory

2. Add the following line into $SAHI_HOME/userdata/config/userdata.properties
debug.traffic.log.unmodified=true

3. Create a soft link of the googlemusic directory in $SAHI_HOME/userdata/scripts
(e.g.  ln sf ~/work/testtool/personal/googlemusic/ )
